ISTRUZIONI PER L'ICONA DELLA BARRA DEI MENU

Per far funzionare correttamente l'app, è necessario aggiungere un'icona template in formato PNG chiamata "IconTemplate.png" in questa cartella.

Requisiti dell'icona:
1. Formato PNG trasparente
2. Solo pixel bianchi o trasparenti (non colorati)
3. Dimensione consigliata: 22x22 pixel (retina: 44x44 pixel)
4. Nome file: IconTemplate.png

Puoi scaricare icone template gratuite dai seguenti siti:
- https://www.flaticon.com (cerca "menu bar icons")
- https://icons8.com
- https://www.iconfinder.com

Altrimenti, puoi utilizzare un'icona predefinita di macOS modificando il percorso nel file main.js. 

# Note sull'Icona

Questo file IconTemplate.png è un template per l'icona della menubar in macOS.

Segui queste linee guida:
- L'icona deve essere principalmente in bianco e trasparente
- La dimensione consigliata è 22x22 pixel
- L'icona dovrebbe essere semplice e facilmente riconoscibile

# Screenshot

Per aggiungere uno screenshot dell'applicazione:
1. Cattura uno screenshot dell'app in funzione
2. Salvalo come "screenshot.png" in questa cartella
3. Esegui git add src/assets/screenshot.png
4. Esegui git commit -m "Aggiunto screenshot dell'applicazione"

Il link al file nel README.md è già impostato per puntare a src/assets/screenshot.png
